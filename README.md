# Fridge23 - Food Rescue App

Welcome to Fridge23! This application is designed to help you reduce food waste, save money, and manage your kitchen inventory effectively.

## Project Info

Fridge23 allows users to:
- Track pantry inventory
- Discover recipes based on available ingredients
- Scan receipts and products (simulated)
- Track food rescue impact and gamification stats
- Connect with a community of eco-conscious cooks

## Getting Started

To run this project locally, ensure you have **Node.js** & **npm** installed.

1.  **Install dependencies:**
    ```sh
    npm install
    ```

2.  **Start the development server:**
    ```sh
    npm run dev
    ```

## Technology Stack

This project is built with modern web technologies:

- **Vite** - Build tool and dev server
- **TypeScript** - Type safety
- **React** - UI Library
- **shadcn/ui** - Component library
- **Tailwind CSS** - Styling
- **TanStack Query** - Data fetching/state management
- **React Router** - Navigation

## Deployment

You can deploy this project to any static site host (Vercel, Netlify, GitHub Pages, etc.).

Build the project for production:
```sh
npm run build
```

The output will be in the `dist` folder.
